# OmniSearch Synthesis Engine

Welcome to the **OmniSearch Synthesis Engine** skill module for the **GenPark/OpenClaw** ecosystem! 🚀

This skill empowers your agent with the ability to dynamically synthesize multi-source information into cohesive, beautifully structured report pages. Instead of merely retrieving fragmented search results, your agent can deliver complete, deeply researched "knowledge pages" on demand.

## 🛠️ Installation & Usage
Clone this repository into your agent framework's `skills/` directory.

```bash
cd your-agent-workspace/skills
git clone https://github.com/alphaparkinc/openclaw-omnisearch-synthesis.git
```

## Features
- **Parallel Data Aggregation**: Access multiple sources simultaneously.
- **Deep Synthesis**: Condense knowledge into standalone pages.
- **Dynamic Curation**: Provide well-formatted, categorization-ready content tailored to the user's intent.
